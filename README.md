## 프로젝트 동기 :hammer:
> COVID-19 확산 및 언택트 소비 증가에 따라 인테리어 매장에 가지 않아도 증강현실로 제품을 직접 확인하고 구매할 수 있는 것을 목적으로 함.

<br>

## 프로젝트 소개 📝
> <p> 사용자를 가구사이트(사용자 <-> 가구사이트)에 연결시켜주는 중계 플랫폼이며, 사용자는 AR로 제품을 직접 자신의 방에 배치해보고, 구매 의사를 결정할 수 있음. </p>
> <p> <b>AOS - Java</b> 사용.  
> <p> <b>MVP 패턴</b>을 사용하여 View <-> Model 간 분리를 진행하였고, <b>mySQL + Node.js</b>로 백엔드 서버 구축을 하였음. </p>
> <p> <b>증강현실</b> 사용 위해 Unity + Android 통합 진행.
